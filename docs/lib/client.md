# 客户端类

获取客户端访问相关信息

## 方法

### 获取客户端语言

```php
\dux\lib\Client::getUserLang();
```

### 获取客户端IP

```php
\dux\lib\Client::getUserIp();
```

### 获取访问来源

```php
\dux\lib\Client::getUrlSource();
```

### 获取浏览器 UA 

```php
\dux\lib\Client::getUserAgent();
```

### 获取浏览器名称

```php
\dux\lib\Client::getUserBrowser();
```

### 获取客户端系统

```php
\dux\lib\Client::getUserOs();
```
